## Reddit (r/webdev / r/reactjs)

Title: I got tired of bloated starter templates so I built a zero-bloat frontend toolkit

Body:

Every "starter kit" I tried installed 50+ packages. I'd use maybe 12 of them. The rest just sat there wasting space and sometimes conflicting.

So I built Numb.Design — a free CLI + resource catalog that installs only the packages your project actually needs.

How it works:
1. Run `npx numb-design init`
2. Tell it what you're building (landing page, dashboard, SaaS, etc.)
3. Tell it your style and required features
4. It generates an install plan with exactly those packages

It also has a catalog of 500+ free resources (UI components, icons, fonts, animations, templates) you can search and add individually.

The CLI is agent-friendly too — AI coding tools can read the docs and execute commands automatically.

All free, all open-source (MIT).
