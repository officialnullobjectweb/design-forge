## Key Areas

### Visual Design
- Bold, intentional layouts with asymmetric grids and clear visual hierarchy
- Consistent color systems with semantic meaning
- Typography scales and pairing strategies
- Spacing systems using 4px or 8px base units

### Component Architecture
- Component state management (rest, hover, active, focus, disabled, loading, error, empty)
- Composition patterns
- Responsive component design

### Accessibility
- WCAG 2.2 AA compliance
- Color contrast requirements
- Focus management
- Screen reader support

### Performance
- Image optimization (lazy loading, WebP, responsive srcset)
- CSS animation optimization (GPU-accelerated properties)
- Code splitting and lazy loading
- Font loading best practices
